# Cuckoo

电影票房分析工具

## Running

``` bash
git clone https://github.com/xiaoyuhen/Cuckoo.git
cd Cuckoo
npm install
npm start
```

Cuckoo 支持通过yarn安装~

``` bash
git clone https://github.com/xiaoyuhen/Cuckoo.git
cd Cuckoo

// 安装yarn，安装失败请加sudo
npm install yarn -g

// 通过yarn安装依赖包
yarn install

// 启动项目
yarn start

```
